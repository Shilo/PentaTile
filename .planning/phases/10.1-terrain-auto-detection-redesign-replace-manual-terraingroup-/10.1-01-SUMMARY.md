## Decisions Made

- Used snapshot-before-clear for name/color persistence: snapshot old names/colors BEFORE clearing terrain_set 0; after recreating, restore names that don't start with "Terrain " (user-set) and non-hash colors
- Used `hash("tc%d" % i) % 256` for deterministic color generation (same seed → same color every rebuild)
- Dual-grid dispatch uses return after per-corner painting to skip single-grid path (clear early-return pattern)
- Terrain_id passed as strip_index to mask_to_atlas for both dual-grid (per-corner tid) and single-grid (terrain_id variable) dispatch
