## Ranking Rationale

1. **Terrain + Variation Authoring Research Spike** is highest because it directly addresses the refactor risk before code commits the project to a candidate-index shape.
2. **Art Quality Pack** is valuable, but variation is now design-coupled to terrain and should not be implemented as a shortcut.
3. **Adoption / UX Pack** is the cleanest low-risk alternate if the spike is postponed.
4. Tilesetter, tooling, performance, and distribution remain useful but have clearer blockers or lower urgency.

## Dependency Notes

- `VAR-01` and `VAR-PIXEL-01` should not choose a final seed/property/API shape until the spike resolves how variation banks coexist with terrain banks, alternatives, and multiple atlas sources.
- `MULTITERR-01..08` should not become production work until the spike defines real Godot-authored TileSet fixtures and the user completes outside-repo manual testing.
- `TOP-01` remains independent enough to implement later, but it should stay explicit per mask/layout and avoid "tile below" heuristics.
- `PERF-02` is a benchmark prerequisite, not permission for persistent caches, worker managers, or shader paths.
