This plan executes a hybrid sequence: agent-runnable git commands sandwich a **manual user gate** (GitHub UI rename) and a **manual user step** (close IDE, rename local directory, reopen IDE).

**Order of operations (do not reorder):**

1. **CHECKPOINT-A** (manual user gate) — User renames the GitHub repo via the GitHub web UI.
2. **Agent step** — `git remote set-url origin <new-url>` + `git remote -v` + `git fetch origin` + `git status`.
3. **CHECKPOINT-B** (manual user gate) — User closes Godot editor + IDE/terminals, renames local directory, reopens.
4. **Agent step** — Migrate Claude memory directory via `mv`.
5. **CHECKPOINT-C** (manual user verify) — Open Godot at the new path, confirm demo loads + paints, confirm `git pull` round-trip works.
6. **Agent step** — Final commit + CHANGELOG sanity check.

The plan is **non-autonomous** because Steps 1, 3, and 5 require human action that can't be CLI-automated.
</wave_5_orchestration>
