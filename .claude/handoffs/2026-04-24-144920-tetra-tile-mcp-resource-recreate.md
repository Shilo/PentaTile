## Codebase Understanding

### Architecture Overview

`TetraTileMapLayer` is the one public runtime class. It extends `TileMapLayer`, overrides `_update_cells(coords, forced_cleanup)`, samples four logic cells around each visual coordinate, and writes transformed atlas cells to internal managed visual layers. The two true diagonal masks (`6` and `9` with bit order `TL=1, TR=2, BL=4, BR=8`) require composition on an overlay layer.
